# College Society Member Hub #

A modern Android application built with **Kotlin**, **Jetpack Compose**, and **MVVM architecture** to manage members and announcements within a college society.

---

## Key Features:

- Role-based access: **Admin** and **Member** (mocked at app start)
- Member directory with role and domain information
- Admin-only add, edit, and delete functionality
- Announcements with **priority levels** (High, Medium, Low)
- Priority-based sorting and filtering
- Clean, responsive UI using Material 3

---

## Architecture & Tech:

- MVVM (Model–View–ViewModel)
- Jetpack Compose
- StateFlow
- Kotlin
- Material 3

---

## App Flow:

1. Select role (Admin / Member)
2. Access dashboard with Members and Announcements
3. Admin manages data; Members have read-only access
